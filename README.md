🚀 Conceptify Frontend - Digital Showcase
Bienvenue dans le dépôt frontend de Conceptify, une plateforme digitale haut de gamme spécialisée dans le développement web, mobile et les solutions IA. Cette interface est conçue pour offrir une expérience utilisateur fluide, luxueuse et interactive.

💎 Caractéristiques du Design
Style Luxe & Minimaliste : Utilisation de glassmorphism, d'éléments 3D et d'une palette sombre élégante.

Animations Fluides : Intégration de motion/react (Framer Motion) pour des transitions de pages et des révélations au scroll fluides.

Interface Responsive : Adaptée pour tous les écrans, du mobile au desktop, avec un menu de navigation intelligent.

Tableau de Bord Administrateur : Interface privée sécurisée pour la gestion des portfolios, références et projets.

🛠️ Stack Technique
Framework : React (Vite)

Langage : TypeScript

Stylisation : Tailwind CSS

Animations : Motion/React

Icônes : Lucide React

Gestion API : Axios

📂 Structure du Projet
Plaintext
src/
 ┣ api/           # Configuration Axios
 ┣ assets/        # Ressources statiques (images, logos)
 ┣ components/    # Composants réutilisables (Footer, Navbar, Sidebar, etc.)
 ┣ pages/         # Vues principales (Home, Portfolio, Library, Dashboard)
 ┣ App.tsx        # Configuration des routes
 ┗ main.tsx       # Point d'entrée
⚙️ Installation et Configuration
Cloner le dépôt

Bash
git clone https://github.com/votre-compte/conceptify-frontend.git
cd conceptify-frontend
Installer les dépendances

Bash
npm install
Configuration des variables d'environnement
Créez un fichier .env à la racine et configurez l'URL de votre backend :

مقتطف الرمز
VITE_API_URL=http://localhost:8000/api
Lancer l'application en mode développement

Bash
npm run dev
